  • setAnimate(value: boolean): void
    • Defined in bootprompt.ts:1016

    Set the animate flag. When the flag is on, the modals produced by this library are animated. When off, they are not animated.

    NOTE: The reason this function exists is to be able to turn off animations during testing. Having the animations turned on can turn simple tests into complicated afairs because it takes a while for a modal to show up or be removed. We do not recommend using this function in production.
